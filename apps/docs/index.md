## 网络部分目标

1. 说清楚一次 HTTP 请求在协议栈中的分层流转。
2. 解释 TCP 建连、可靠传输、流量控制、拥塞控制的关系。
3. 讲明白 IP/ARP/DNS/路由器分别在解决什么问题。
4. 对比 HTTP1.1/2/3 与 HTTPS/TLS 的核心改进点。
