# Funny Game

接食物小游戏（原生 HTML/CSS/JavaScript）。

## 玩法

- 玩家在屏幕底部左右移动接住食物。
- 接到食物加分。
- 接到炸弹扣生命，护盾可抵挡一次。
- 每 100 分提升难度。
- 支持键盘和触屏按钮。

## 道具

- magnet
- slow
- double
- shield
- clear
- reverse

## 本地运行

1. 安装依赖

```bash
npm install
```

2. 启动静态服务

```bash
npm run serve
```

3. 打开 `http://127.0.0.1:4173/index.html`
